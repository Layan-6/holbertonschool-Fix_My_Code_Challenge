# Fix My Code Challenge

This project is part of the Holberton School Fix My Code Challenge.

## Task
Debug and fix the FizzBuzz implementation in Python.

## File
- `0-fizzbuzz.py`

## Issue Fixed
The program was printing `Fizz` instead of `FizzBuzz` for numbers divisible by both 3 and 5.

## Solution
The condition for numbers divisible by both 3 and 5 was moved before the checks for 3 or 5 individually.
